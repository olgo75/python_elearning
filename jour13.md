# Jour 13 : Les listes en compréhension

## Objectifs
- Découvrir la compréhension de liste

## 1. Syntaxe
```python
carres = [x * x for x in range(1, 6)]
print(carres)
```

## 2. Avantages
- Plus court et lisible
- Permet de filtrer ou transformer facilement
